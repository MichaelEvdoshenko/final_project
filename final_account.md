# Крестики-нолики с ИИ: Итоговый отчет

## О проекте
Проект реализует классическую игру "Крестики-нолики" с поддержкой различных алгоритмов ИИ. Поддерживаются доски размером от 3x3 до 5x5, два типа ботов (Q-learning и MCTS), а также пользовательский интерфейс.

## Проделанная работа

### Работа над игровой логикой и алгоритмами ИИ
1. Реализованы два типа ботов
2. Эмпирически подобраны оптимальные гиперпараметры для Q-learning
3. Добавлены эвристики для улучшения игры ботов

**Q-learning бот:**
1. Использует табличный метод обучения
2. Оптимальные параметры: `alfa=0.1`, `eps=0.9`
3. Показал хорошие результаты только на доске 3x3
4. **Ключевое замечание**: Предобучение неэффективно - загрузка Q-таблицы занимает столько же времени, сколько обучение с нуля
5. В папке /models оставлены предобученные таблицы в качестве доказательства

**MCTS бот (Monte Carlo Tree Search):**
1. Использует метод Монте-Карло для поиска по дереву решений
2. 10000 симуляций на ход для баланса качества/скорости
3. Добавлена эвристика блокировки выигрышных ходов противника
4. **Ключевое замечание**: Значительно превосходит Q-learning на досках 4x4 и 5x5

**Выводы по алгоритмам:**
1. Q-learning эффективен только для маленьких значений таблицы (3x3)
2. MCTS лучше и показывает хороший результат на домках любого размера
3. Эвристики для MCTS не необходимы, а лишь помогают на начальных этапах игры, так как бот не учитывает качество ходов игрока

**Основные компоненты UI:**
1. Стартовый экран с выбором размера доски, типа бота, игры с ботом\игроком
2. Система отображения текущего состояния игры
